# Ensemble Learning for Precision Agriculture

## Problem

Recommend the most suitable crop based on soil and environmental conditions.

## Dataset

Features:
- Nitrogen
- Phosphorus
- Potassium
- Temperature
- Humidity
- pH
- Rainfall

Target:
- Crop Label

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Model Training
4. Hyperparameter Tuning
5. Evaluation
6. Interpretation

## Models

- Random Forest
- XGBoost

## Results

Random Forest:
F1 Score = 0.9932

XGBoost:
F1 Score = 0.9885

## Key Findings

- Rainfall was among the most influential features.
- Ensemble methods achieved excellent predictive performance.
- Random Forest slightly outperformed XGBoost.
